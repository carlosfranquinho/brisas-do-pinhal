      Evolução Climatológica

      Acerca da Estação

      Hardware, localização e notas sobre os dados.

      Hardware

      Modelo
      Froggit HP2000
      Fabricante
      Froggit (protocolo Ecowitt)
      Sensores incluídos
      Temperatura e humidade exterior, anemómetro (velocidade e direção), pluviómetro de báscula, barómetro, radiação solar e índice UV
      Transmissão
      Sem fios 915 MHz (sensor externo → consola)
      Envio de dados
      Protocolo Ecowitt via Wi-Fi, a cada 60 segundos

      Localização

      Local
      Marinha Grande, Leiria, Portugal
      Enquadramento
      Pinhal do Rei (Pinhal de Leiria) — floresta litoral atlântica
      Coordenadas
      39° 45' N, 8° 56' W
      Altitude
      ~50 m
      Referência aeronáutica
      METAR: LPMR (Base Aérea de Monte Real, ~12 km)

      Dados e Arquivo

      Início do registo local
      20 de abril de 2021
      Período com lacunas
      Maio–agosto 2025 (estação offline por obras no edifício); lacunas pontuais em 2023–2024
      Preenchimento de lacunas
      Dias sem dados são complementados com registos de estações Weather Underground vizinhas (IPATAI5, IBARRE4, entre outras), identificados na base de dados pela estação de origem
      Armazenamento
      Base de dados SQLite local; API FastAPI exposta via Cloudflare Tunnel
      Normais climatológicas
      Série 1991–2020 da estação de Alcobaça — Estação de Fruticultura Vieira Natividade (IPMA)

      Módulo exterior Froggit HP2000 — sensores de temperatura, humidade, vento, chuva, UV e radiação solar.

      Notas

      Os valores apresentados são medições de uma estação meteorológica pessoal e não têm caráter oficial. A estação não é calibrada periodicamente por entidade certificada. Os dados de temperatura de 1 de janeiro a 7 de março de 2021 provêm da estação IBARRE4 (Weather Underground), com correção estatística de viés aplicada para aproximar os valores às condições locais.
